# Matrix Ticket System

A small learning project that explores how to work with the Matrix API.

The goal is to create a bot that can open support tickets for a specific user or a group of users.  
Tickets are displayed in a WebView where they can be managed (e.g., add comments or close the ticket).

When a ticket is closed, the bot automatically sends a notification message to the user who created the ticket.
